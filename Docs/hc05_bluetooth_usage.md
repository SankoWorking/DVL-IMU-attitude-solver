# HC-05 蓝牙串口使用说明

本文按当前 H743 工程配置整理。HC-05 在本项目里作为经典蓝牙 SPP 串口透传模块使用；蓝牙命令口使用独立 `UART8 PE1/PE0`，调试日志和有线命令口使用 `USART1 PA9/PA10`。

## 1. 当前串口分配

| 用途 | STM32 串口 | 引脚 | 说明 |
| --- | --- | --- | --- |
| 调试日志/有线命令口 | USART1 | PA9 TX / PA10 RX | CH340 调试日志、传感器周期输出和有线命令输入 |
| HC-05 蓝牙命令口 | UART8 | PE1 TX / PE0 RX | 手机蓝牙串口发送控制命令并接收 `[BT]` 回复 |

USART2、UART4、UART5、USART6 当前给 4 个声纳使用，USART3 给 VLS-H5，UART7 给 TF-UW500，所以不要再拿这些口接 HC-05。

## 2. 正常运行接线

正常透传时 `KEY/EN` 不接，或者保持低电平。只接 UART、电源和地：

| HC-05 引脚 | STM32H743 引脚 | 说明 |
| --- | --- | --- |
| `TXD` | `PE0 / UART8_RX` | HC-05 发给 STM32 的命令 |
| `RXD` | `PE1 / UART8_TX` | STM32 返回 `[BT]` 帮助/OK |
| `GND` | `GND` | 必须共地 |
| `VCC` | 按模块底板标注供电 | 常见 HC-05 底板可接 5V，裸模块用 3.3V |
| `KEY/EN` | 不接 | 正常手机透传不用接 |

当前固件里 HC-05 使用的 `UART8` 是 `115200 8N1`。

## 3. AT 配置接线

AT 模式是配置 HC-05 自己的参数，推荐先用 USB-TTL 单独配置模块，不经过 STM32：

```text
USB-TTL TXD -> HC-05 RXD
USB-TTL RXD -> HC-05 TXD
USB-TTL GND -> HC-05 GND
HC-05 VCC  -> 按模块底板标注供电
HC-05 KEY/EN -> 3.3V
```

进入 AT 模式的常见方式：

1. 上电前按住 HC-05 底板按键，或先把 `KEY/EN` 拉到 3.3V，再给 HC-05 上电；AT 波特率通常是 `38400`。
2. 有些模块支持正常上电后再拉高 `KEY/EN`，这时 AT 波特率可能是当前数据波特率，出厂常见 `9600`。

串口调试助手选择对应 AT 波特率，发送命令时带 `\r\n`：

```text
AT
AT+ROLE=0
AT+NAME=H743_CAR
AT+PSWD=1234
AT+UART=115200,0,0
AT+RESET
```

每条成功通常返回 `OK`。如果 `AT+PSWD=1234` 不成功，换成 `AT+PSWD="1234"` 再试。配置完成后断电，断开 `KEY/EN`，再按第 2 节接到 STM32 的 `UART8` 正常运行。

## 4. 手机连接

1. HC-05 正常上电，不进入 AT 模式。
2. 手机蓝牙里搜索并配对 `H743_CAR` 或模块当前名称。
3. 配对码通常是 `1234`。
4. 打开支持经典蓝牙 SPP 的串口 App，连接 HC-05。
5. 发送 `?`，应返回命令帮助。

蓝牙口只承担命令交互；周期传感器日志继续由 `USART1 PA9/PA10` 输出。
手动 `M/A/B/C/D/0` 命令会退出自动巡航，不会被 VLS-H5 前后避障清零；VLS-H5 前后 `400mm` 保护只在自动巡航 `X` 开启后生效。

## 5. 发送命令

手机蓝牙串口发送的命令会进入 `UART8_rxCallback()`，再进入 `App_Process_Command(APP_BLUETOOTH_UART, ...)`。有效控制命令会从蓝牙口返回 `[BT] OK`。

| 命令 | 作用 |
| --- | --- |
| `?` 或 `H` | 返回命令帮助 |
| `M,-15,-15` | 左右主电机小功率前进 |
| `M,15,15` | 左右主电机小功率后退 |
| `M,0,0` 或 `0` | 左右主电机停止 |
| `!` | 顶部电机打开 |
| `Q` 或 `%` | 顶部电机停止 |
| `A` / `B` / `C` / `D` | 前进、后退、右转、左转 |
| `3` / `6` | 全局速度加 10 / 减 10 |
| `Z` | MPU6500 yaw 清零 |
| `X` / `Y` | 自动巡航开启 / 关闭并停车 |

调电机时先架空车体，先发 `0` 或 `M,0,0`，再发小功率命令。

## 6. 常见问题

| 现象 | 处理 |
| --- | --- |
| 能配对但发 `?` 没返回 | HC-05 数据波特率可能还是 9600，重新进 AT 模式查 `AT+UART?` |
| App 显示乱码 | 确认 HC-05 和 `UART8` 都是 `115200 8N1` |
| 手机发送无反应 | 检查 HC-05 `TXD -> PE0`、`RXD -> PE1` 是否接反，以及 GND 是否共地 |
| 一直进 AT 模式，手机连不上 | `KEY/EN` 还拉着高电平；正常运行时断开 `KEY/EN` |
| 想看传感器日志 | 连接 CH340 到 `USART1 PA9/PA10` 调试口查看 |
